# Dispenser

## Features

- Easy administration
- Domain reporting
- Easy configuration
- Fast

## Coming soon
- Prevent user from getting the same domain twice

## Commands 
<table>
<thead>
<tr>
<th>Command</th>
<th>Description</th>
<th>Usage</th>
</tr>
</thead>
<tbody>
<tr>
<td>/add</td>
<td>Add a link to the Database</td>
<td>/add example.com &lt;-- do not include https://</td>
</tr>
<tr>
<td>/remove</td>
<td>Remove a link from the Database</td>
<td>/remove example.com</td>
</tr>
<tr>
<td>/panel</td>
<td>Creates a panel of buttons to click for the links!</td>
<td>/panel</td>
</tr>
<tr>
<td>/list</td>
<td>Lists all domains stored in the Database</td>
<td>/list</td>
</tr>
<tr>
<td>/reset-all</td>
<td>Resets all users in the database</td>
<td>/reset</td>
</tr>
<tr>
<td>/reset</td>
<td>Reset a user</td>
<td>/reset [user]</td>
</tr>
</tbody>
</table>
